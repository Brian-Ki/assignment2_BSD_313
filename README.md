# Tribute Page Project: Kobe Bryant

Welcome to the **Tribute Page** repository for the BSD 313 assignment! This project is a responsive, semantic webpage dedicated to the life, legacy, and "Mamba Mentality" of basketball legend Kobe Bryant.

---

## Features

* **Semantic HTML5 Layout:** Structured cleanly using modern elements like `<header>`, `<main>`, `<section>`, and `<footer>` for optimal readability and SEO compliance.
* **Custom CSS Styling:** Features a polished layout using the iconic Los Angeles Lakers color scheme (Purple & Gold).
* **CSS Keyframe Animation:** Includes a smooth, self-contained CSS animation of a basketball bouncing and rolling across the screen from right to left without disrupting the page content.
* **Fully Responsive:** Designed to look crisp and readable across various screen sizes, from mobile phones to desktops.

---

## Built With

* **HTML5** - Page structure and semantics
* **CSS3** - Page layout, typography, and keyframe animations
